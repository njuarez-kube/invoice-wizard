The Invoice Wizard

Installation and usage guide — follow these steps in order.

Part 1 — Install Python (first time only)

The app runs on Python. You only need to do this once, ever.

1
Open your browser and go to this address:

https://www.python.org/downloads/

Click the big yellow Download Python button. A file will download.

2
Run the installer

Open the downloaded file (it looks like python-3.x.x-amd64.exe).

⚠ Very important: Before clicking anything, find the checkbox at the bottom of the installer window that says "Add Python to PATH" — make sure it is checked (ticked). If you miss this step the app will not work.

Then click Install Now and wait for it to finish. Click Close when done.

Part 2 — Install the app (first time only)

3
Extract the ZIP file

Find the invoice-processor.zip file you received.

Right-click on it → Extract All… → choose a location (for example your Desktop) → click Extract.

You will get a folder called invoice-processor. Open it.

4
Double-click install.bat

A black window will appear and show text scrolling by. This is normal — it is downloading the required components.

Wait until it says "All requirements are installed. You are ready to go!" then press any key to close the window.

✓ You only need to do this once. Next time, skip straight to Part 3.

Part 3 — Using the app every day

5
Create a Desktop shortcut (optional, recommended)

Double-click create-shortcut.bat inside the app folder.

A shortcut called The Invoice Wizard will appear on your Desktop. From now on, just double-click that icon to start the app — no need to open the folder each time.

✓ Every time you want to use the app:

Double-click the The Invoice Wizard icon on your Desktop.

(Or double-click start.bat inside the app folder if you did not create the shortcut.)

A black window will appear and stay open — do not close it. Your browser will open automatically with the app.

When you are done, close the black window to stop the app.

✓ Not sure if everything is installed correctly?
Double-click check.bat at any time. It will show a green [OK] or red [MISSING] for each requirement, and tell you exactly what to do.
ℹ If the browser does not open automatically:
Open your browser manually and go to: http://localhost:8080

Part 4 — Google Drive vendor sync

The app connects to a shared Google Drive folder where all vendor invoice templates are stored. From the Setup page you can keep your local copy in sync.

↓
Update — download the latest templates

Click Update to download any new or changed vendor templates from Drive. Do this whenever you start a new session or a colleague tells you they have added a new vendor.

↑
Add to Drive — share a new vendor you created

If you set up a new vendor on the Setup page, click Add to Drive to upload it so everyone else can download it with their next Update.

✓ Typical workflow: Click Update at the start of your day → process invoices → if you created a new vendor, click Add to Drive before closing the app.
ℹ The green Connected chip next to "Google Drive Sync" confirms the app has access. If it shows Disconnected, contact your administrator.

Troubleshooting

How to open Command Prompt (CMD) manually

Sometimes support staff may ask you to open a terminal window. Here is how:

Easiest way — from the app folder:

  • Open the invoice-processor folder in File Explorer
  • Click on the address bar at the top (it shows the folder path)
  • Type cmd and press Enter
  • A black window opens, already inside the correct folder

Alternative — from the Start menu:

  • Press ⊞ Win + R
  • Type cmd and press Enter

Alternative — from the taskbar search:

  • Click the search bar next to the Start button
  • Type cmd
  • Click Command Prompt

Common problems

The black window closes immediately when I double-click install.bat or start.bat
Python is probably not installed correctly. Repeat Part 1 and make sure the "Add Python to PATH" checkbox is ticked.

The browser says "This site can't be reached"
The black window from start.bat was closed. Double-click start.bat again and keep the window open.

Port 8080 is already in use
Another program is using that port. Open config.ini with Notepad, change port = 8080 to port = 8081, save, and start again. Then go to http://localhost:8081.

install.bat says "pip is not recognized"
Python was installed without PATH. Uninstall Python (via Settings → Apps), then reinstall it following Part 1 carefully.
